# Yoda Theme for Oh My Zsh

![Theme Preview](../../assets/oh-my-zsh-preview.png)

### Usage

1. Copy the theme file to your Oh My Zsh custom themes directory:

   ```bash
   cp yoda.zsh-theme ~/.oh-my-zsh/custom/themes/
   ```

2. Set the theme in your `~/.zshrc`:

   ```bash
   ZSH_THEME="yoda"
   ```

3. Reload your shell:
   ```bash
   source ~/.zshrc
   ```
